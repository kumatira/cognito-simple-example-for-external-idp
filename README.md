# cognito-simple-example-for-external-idp

すでにcognitoの構築が済んでいる場合はCloudFormationによる構築はスキップ。  
`/public/js/amplifyConfig.js.example` を `/public/js/amplifyConfig.js`とコピーし、必要な情報をセットする。

↓でexpressを起動
```
$ npm install --production
$ npm start
```
